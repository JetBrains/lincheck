# Lincheck Template Project

## Requirements

- **JDK**: JDK 21 or higher
- **Kotlin**: API version 2.2 (compiler version 2.2.20)
- **IDE**: IntelliJ IDEA

## Setup

1. Clone the repository
2. Configure your IntelliJ IDEA to use Java 21+ SDK
3. The project will automatically use Kotlin 2.2.20 compiler settings
4. Import the project and let Gradle sync the dependencies
